# Raffle contract on the Skale chain

## Quickstart
```
git clone https://github.com/Gouldian0120/raffle_skale.git
cd raffle_skale
yarn
```

Deploy:

```
npx hardhat run scripts/chaos.deploy.js --network chaos
```

Testing:

```
npx hardhat test tests/chaos.test.js --network chaos
```
